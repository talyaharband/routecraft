Routecraft Constraints Pipeline

Constraint-first variant for const.py, configured by constraints_parameters_const.json.

  1. Clean raw order/address data and preserve due dates
  2. Geocode and assign priority from required_delivery_date
  3. Select only the loadable orders for the current work arrangement
  4. Cluster nearby selected orders
  5. Create k-means groups and maps
  6. Build per-group distance matrices
  7. Plan each driver sequentially, including warehouse reload segments when useful
  8. Save leftovers for the next run
