## Postal Code Format
Israeli postal codes (mikud) are 7 digits since 2013 (formerly 5 digits).
Format: XXXXXXX (e.g., 6688312)

## Address Format
Standard: [Street] [Number], [City] [Postal Code]
Hebrew: [rechov] [mispar], [ir] [mikud]
